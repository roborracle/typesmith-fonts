# TypeSmith Fonts

Font files for [TypeSmith](https://typesmith.app) - a font discovery and pairing platform.

## Usage

Fonts are served via jsDelivr CDN:

```
https://cdn.jsdelivr.net/gh/roborracle/typesmith-fonts@v1.0.0/external/{slug}/{slug}.woff2
```

## Structure

```
external/           # Font files organized by slug
  {slug}/
    {slug}.woff2    # WOFF2 format (primary)
font_files_index.json  # Index of all fonts with metadata
```
